# Go-To Color Palettes + Usage Rules (Landing Pages)

Pick ONE palette per brand/page. Keep it consistent.

---

## Usage Rules (non-negotiable)

1. **One primary + one accent.** Everything else = neutrals.
2. **Buttons must pop.** Use accent for primary CTA; keep secondary CTA outline.
3. **Contrast first.** Body text must be readable on mobile in sunlight.
4. **Proof > decoration.** Use color to guide attention, not to “look cool.”
5. **Limit gradients.** If used, reserve for hero background only.

## Component Styling Guidance

### Buttons
- Primary CTA: accent color, solid fill, 10–12px radius
- Hover: darken 6–10%
- Secondary CTA: transparent with border (same accent), or “ghost” style

### Sections
- Alternate section background every 1–2 blocks to create rhythm
- Cards: subtle border + light shadow, never heavy skeuomorphism

### Typography
- Headlines: near-black on light, near-white on dark
- Body: never pure black on pure white; use Slate/Gray tones
